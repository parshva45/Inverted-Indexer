     <p>\n       <b>\n        Paleotempestology\n       </b>\n       is the study of past\n       <a href="/wiki/Tropical_cyclone" title="Tropical cyclone">\n        tropical cyclone\n       </a>\n       activity by means of geological proxies as well as historical documentary records. The term was coined by\n       <a href="/wiki/Kerry_Emanuel" title="Kerry Emanuel">\n        Kerry Emanuel\n       </a>\n       .\n      </p>\n      <p>\n      </p>\n      <div class="toc" id="toc">\n id="Methods">\n        Methods\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=Paleotempestology&amp;action=edit&amp;section=1" title="Edit section: Methods">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <h3>\n       <span class="mw-headline" id="Sedimentary_proxy_records">\n        Sedimentary proxy records\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=Paleotempestology&amp;action=edit&amp;section=2" title="Edit section: Sedimentary proxy records">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h3>\n      <p>\n       Examples of proxies include\n       <a href="/wiki/Overwash" title="Overwash">\n        overwash\n       </a>\n       deposits preserved in the\n       <a href="/wiki/Sediment" title="Sediment">\n        sediments\n       </a>\n       of coastal lakes and\n       <a href="/wiki/Marsh" title="Marsh">\n        marshes\n       </a>\n       ,\n       <a href="/wiki/Micropaleontology" title="Micropaleontology">\n        microfossils\n       </a>\n       such as\n       <a href="/wiki/Foraminifera" title="Foraminifera">\n        foraminifera\n       </a>\n       ,\n       <a href="/wiki/Pollen" title="Pollen">\n        pollen\n       </a>\n       ,\n       <a href="/wiki/Diatom" title="Diatom">\n        diatoms\n       </a>\n       ,\n       <a href="/wiki/Dinoflagellate" title="Dinoflagellate">\n        dinoflagellates\n       </a>\n       ,\n       <a href="/wiki/Phytolith" title="Phytolith">\n        phytoliths\n       </a>\n       contained in coastal sediments, wave-generated or flood-generated sedimentary structures or deposits (called\n       <a href="/wiki/Tempestite" title="Tempestite">\n        tempestites\n       </a>\n       ) in marine or\n       <a href="/wiki/Lagoon" title="Lagoon">\n        lagoonal\n       </a>\n       sediments, storm wave deposited\n       <a href="/wiki/Coral" title="Coral">\n        coral\n       </a>\n       shingle, shell, sand and shell and pure sand shore parallel ridges.\n      </p>\n      <p>\n       The method of using overwash deposits preserved in coastal lake and marsh sediments is adopted from earlier studies of paleotsunami deposits. Both\n       <a href="/wiki/Storm" title="Storm">\n        storms\n       </a>\n       and\n       <a href="/wiki/Tsunami" title="Tsunami">\n        tsunamis\n       </a>\n       leave very similar if not identical sedimentary deposits in coastal lakes and marshes and differentiating between the two in a sedimentary record can be difficult. The first studies to examine prehistoric records of tropical cyclones occurred in Australia and the South Pacific during the late 1970s and early 1980s.\n       <sup class="reference" id="cite_ref-Baines1976_1-0">\n        <a href="#cite_note-Baines1976-1">\n         [1]\n        </a>\n       </sup>\n       <sup class="reference" id="cite_ref-Rhodes1980_2-0">\n        <a href="#cite_note-Rhodes1980-2">\n         [2]\n        </a>\n       </sup>\n       <sup class="reference" id="cite_ref-Chappell1983_3-0">\n        <a href="#cite_note-Chappell1983-3">\n         [3]\n        </a>\n       </sup>\n       These studies examined multiple shore parallel ridges of coral shingle or sand and marine shells. As many as 50 ridges can be deposited at a site with each representing a past severe\n       <a href="/wiki/Tropical_cyclone" title="Tropical cyclone">\n        tropical cyclone\n       </a>\n       over the previous 6,000 years. Tsunamis are not known to deposit multiple sedimentary ridges and therefore these features can be more easily attributed to a past storm at any given site.\n      </p>\n      <p>\n       Coastal sedimentary analyses have been done at the\n       <a href="/wiki/Gulf_of_Mexico" title="Gulf of Mexico">\n        U.S. Gulf coast\n       </a>\n       ,\n       <sup class="reference" id="cite_ref-Liu1999_4-0">\n        <a href="#cite_note-Liu1999-4">\n         [4]\n        </a>\n       </sup>\n       <sup class="reference" id="cite_ref-LiuFearn2000_5-0">\n        <a href="#cite_note-LiuFearn2000-5">\n         [5]\n        </a>\n       </sup>\n       the\n       <a class="mw-redirect" href="/wiki/Atlantic" title="Atlantic">\n        Atlantic\n       </a>\n       coast from\n       <a href="/wiki/South_Carolina" title="South Carolina">\n        South Carolina\n       </a>\n       <sup class="reference" id="cite_ref-Scott2003_6-0">\n        <a href="#cite_note-Scott2003-6">\n         [6]\n        </a>\n       </sup>\n       up to\n       <a href="/wiki/New_Jersey" title="New Jersey">\n        New Jersey\n       </a>\n       <sup class="reference" id="cite_ref-Donnelly2001a_7-0">\n        <a href="#cite_note-Donnelly2001a-7">\n         [7]\n        </a>\n       </sup>\n       <sup class="reference" id="cite_ref-Donnelly2004_8-0">\n        <a href="#cite_note-Donnelly2004-8">\n         [8]\n        </a>\n       </sup>\n       and\n       <a href="/wiki/New_England" title="New England">\n        New England\n       </a>\n       ,\n       <sup class="reference" id="cite_ref-Donnelly2001b_9-0">\n        <a href="#cite_note-Donnelly2001b-9">\n         [9]\n        </a>\n       </sup>\n       and the\n       <a href="/wiki/Caribbean_Sea" title="Caribbean Sea">\n        Caribbean Sea\n       </a>\n       .\n       <sup class="reference" id="cite_ref-Donnelly2007_10-0">\n        <a href="#cite_note-Donnelly2007-10">\n         [10]\n        </a>\n       </sup>\n       <sup class="reference" id="cite_ref-McCloskey2008_11-0">\n        <a href="#cite_note-McCloskey2008-11">\n         [11]\n        </a>\n       </sup>\n       Also, studies on pre-historic tropical cyclones hitting\n       <a href="/wiki/Australia" title="Australia">\n        Australia\n       </a>\n       have been made.\n       <sup class="reference" id="cite_ref-Nott2001_12-0">\n        <a href="#cite_note-Nott2001-12">\n         [12]\n        </a>\n       </sup>\n       <sup class="reference" id="cite_ref-Nott2007_13-0">\n        <a href="#cite_note-Nott2007-13">\n         [13]\n        </a>\n       </sup>\n       A study covering the\n       <a href="/wiki/South_China_Sea" title="South China Sea">\n        South China Sea\n       </a>\n       coast has also been published.\n       <sup class="reference" id="cite_ref-14">\n        <a href="#cite_note-14">\n         [14]\n        </a>\n       </sup>\n      </p>\n      <h3>\n       <span class="mw-headline" id="Markers_in_coral">\n        Markers in coral\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=Paleotempestology&amp;action=edit&amp;section=3" title="Edit section: Markers in coral">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h3>\n      <p>\n       Rocks contain certain\n       <a href="/wiki/Isotope" title="Isotope">\n        isotopes\n       </a>\n       of\n       <a href="/wiki/Chemical_element" title="Chemical element">\n        elements\n       </a>\n       , known as natural tracers, which describe the conditions under which they formed. By studying the\n       <a href="/wiki/Calcium_carbonate" title="Calcium carbonate">\n        calcium carbonate\n       </a>\n       in coral rock, past sea surface temperature and hurricane information can be revealed. Heavier\n       <a href="/wiki/Oxygen" title="Oxygen">\n        oxygen\n       </a>\n       isotopes (\n       <a href="/wiki/Oxygen-18" title="Oxygen-18">\n        <sup>\n         18\n        </sup>\n        O\n       </a>\n       ) decrease in relation to lighter oxygen isotopes (\n       <a href="/wiki/Oxygen-16" title="Oxygen-16">\n        <sup>\n         16\n        </sup>\n        O\n       </a>\n       ) in coral during periods of very heavy\n       <a href="/wiki/Rain" title="Rain">\n        rainfall\n       </a>\n       .\n       <sup class="reference" id="cite_ref-15">\n        <a href="#cite_note-15">\n         [15]\n        </a>\n       </sup>\n       Since hurricanes are the main source of extreme rainfall in the tropical oceans\n       <sup class="noprint Inline-Template Template-Fact" style="white-space:nowrap;">\n        [\n        <i>\n         <a href="/wiki/Wikipedia:Citation_needed" title="Wikipedia:Citation needed">\n          <span title="This claim needs references to reliable sources. (December 2016)">\n           citation needed\n          </span>\n         </a>\n        </i>\n        ]\n       </sup>\n       , past hurricane events can be dated to the days of their impact on the coral by looking at the decreased\n       <sup>\n        18\n       </sup>\n       O concentration within the coral.\n       <sup class="reference" id="cite_ref-16">\n        <a href="#cite_note-16">\n         [16]\n        </a>\n       </sup>\n      </p>\n      <h3>\n       <span class="mw-headline" id="Speleothems_and_tree_rings">\n        Speleothems and tree rings\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=Paleotempestology&amp;action=edit&amp;section=4" title="Edit section: Speleothems and tree rings">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h3>\n      <p>\n       Isotope studies in\n       <a href="/wiki/Speleothem" title="Speleothem">\n        speleothems\n       </a>\n       and tree rings (\n       <a href="/wiki/Dendrochronology" title="Dendrochronology">\n        dendrochronology\n       </a>\n       ) offers a means by which higher resolution records of long-term tropical cyclone histories can be attained.\n       <sup class="reference" id="cite_ref-17">\n        <a href="#cite_note-17">\n         [17]\n        </a>\n       </sup>\n       Unlike the isotope records, the sedimentary records are too coarse in their resolution to register quasi-cyclic activity at decadal to centennial scales. These higher resolution records therefore offer a means for possibly differentiating between the natural variability of tropical cyclone behaviour and the effects of anthropogenically induced global\n       <a href="/wiki/Climate_change" title="Climate change">\n        climate change\n       </a>\n       . Recent studies with\n       <a href="/wiki/Stalagmite" title="Stalagmite">\n        stalagmites\n       </a>\n       in\n       <a href="/wiki/Belize" title="Belize">\n        Belize\n       </a>\n       shows that events can be determined on a week-by-week basis.\n       <sup class="reference" id="cite_ref-Frappier2007_18-0">\n        <a href="#cite_note-Frappier2007-18">\n         [18]\n        </a>\n       </sup>\n      </p>\n      <h3>\n       <span class="mw-headline" id="Historical_records">\n        Historical records\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=Paleotempestology&amp;action=edit&amp;section=5" title="Edit section: Historical records">\n </div>\n      <p>\n       Before the invention of the\n       <a href="/wiki/Telegraphy" title="Telegraphy">\n        telegraph\n       </a>\n       in the early to mid-19th century, news was as fast as the fastest horse or stagecoach or ship. Normally, there was no advance warning of a tropical cyclone impact. However, the situation changed in the 19th century as seafaring people and land-based researchers, such as Father Vi\xc3\xb1es in\n       <a href="/wiki/Cuba" title="Cuba">\n        Cuba\n       </a>\n       , came up with systematic methods of reading the sky\'s appearance or the sea state, which could foretell a tropical cyclone\'s approach up to a couple days in advance.\n      </p>\n      <p>\n       One of the best documented storms is the\n       <a href="/wiki/Royal_Charter_Storm" title="Royal Charter Storm">\n        Royal Charter Storm\n       </a>\n       of 1859 which caused over 800 deaths in the UK alone. It led directly to the formation of the\n       <a class="mw-redirect" href="/wiki/Meteorological_Office" title="Meteorological Office">\n        Meteorological Office\n       </a>\n       under\n       <a class="mw-redirect" href="/wiki/Robert_Fitzroy" title="Robert Fitzroy">\n        Robert Fitzroy\n       </a>\n       . However, wind speed could not be measured accurately at the time, methods only becoming available after the\n       <a href="/wiki/Tay_Bridge_disaster" title="Tay Bridge disaster">\n        Tay Bridge disaster\n       </a>\n       of 1879. One of the better sources for storms in and around\n       <a href="/wiki/Great_Britain" title="Great Britain">\n        Britain\n       </a>\n       is the\n       <a href="/wiki/Shipwreck" title="Shipwreck">\n        shipwreck\n       </a>\n       statistics compiled annually by the\n       <a class="mw-redirect" href="/wiki/Board_of_Trade" title="Board of Trade">\n        Board of Trade\n       </a>\n       , but which have yet to be analysed in detail.\n      </p>\n      <p>\n       Michael Chenoweth used 18th century journals to reconstruct the climate of\n       <a href="/wiki/Jamaica" title="Jamaica">\n        Jamaica\n       </a>\n       .\n       <sup class="reference" id="cite_ref-Chenoweth2003_19-0">\n        <a href="#cite_note-Chenoweth2003-19">\n         [19]\n        </a>\n       </sup>\n       Together with Dmitry Divine, he also created a 318-year (1690\xe2\x80\x932007) record of tropical cyclones in the\n       <a href="/wiki/Lesser_Antilles" title="Lesser Antilles">\n        Lesser Antilles\n       </a>\n       , using newspaper accounts, ships\'\n       <a href="/wiki/Logbook" title="Logbook">\n        logbooks\n       </a>\n       , meteorological journals, and other document sources.\n       <sup class="reference" id="cite_ref-20">\n        <a href="#cite_note-20">\n         [20]\n        </a>\n       </sup>\n      </p>\n      <p>\n       In\n       <a href="/wiki/China" title="China">\n        China\n       </a>\n       , the abundance of historical documentary records in the form of\n       <i>\n        Fang Zhi\n       </i>\n       (semiofficial local\n       <a href="/wiki/Gazette" title="Gazette">\n        gazettes\n       </a>\n       ) offers an extraordinary opportunity for providing a high-resolution historical dataset for the frequency of\n       <a class="mw-redirect" href="/wiki/Pacific_typhoon" title="Pacific typhoon">\n        typhoon\n       </a>\n       strikes.\n       <sup
