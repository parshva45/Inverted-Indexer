1924MNRAS..84..308E\n        </td>\n       </tr>\n      </table>\n      <p>\n       The\n       <b>\n        bibcode\n       </b>\n       (also known as the\n       <b>\n        refcode\n       </b>\n <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=Bibcode&amp;action=edit&amp;section=1" title="Edit section: Adoption">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <p>\n       The Bibliographic Reference Code (refcode) was originally developed to be used in\n       <a href="/wiki/SIMBAD" title="SIMBAD">\n        SIMBAD\n       </a>\n       and the\n       <a href="/wiki/NASA/IPAC_Extragalactic_Database" title="NASA/IPAC Extragalactic Database">\n        NASA/IPAC Extragalactic Database\n       </a>\n       (NED), but it became a de facto standard and is now used more widely, for example, by the NASA\n       <a href="/wiki/Astrophysics_Data_System" title="Astrophysics Data System">\n        Astrophysics Data System\n       </a>\n       who coined and prefer the term "bibcode".\n       <sup class="reference" id="cite_ref-a_1-0">\n        <a href="#cite_note-a-1">\n         [1]\n        </a>\n       </sup>\n       <sup class="reference" id="cite_ref-b_2-0">\n        <a href="#cite_note-b-2">\n         [2]\n        </a>\n       </sup>\n      </p>\n      <h2>\n       <span class="mw-headline" id="Format">\n        Format\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=Bibcode&amp;action=edit&amp;section=2" title="Edit section: Format">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <p>\n       The code has a fixed length of 19 characters and has the form\n      </p>\n      <center>\n       <tt>\n        YYYYJJJJJVVVVMPPPPA\n       </tt>\n      </center>\n      <p>\n       where\n       <tt>\n        YYYY\n       </tt>\n       is the four-digit year of the reference and\n       <tt>\n        JJJJJ\n       </tt>\n       is a code indicating where the reference was published. In the case of a journal reference,\n       <tt>\n        VVVV\n       </tt>\n       is the volume number,\n       <tt>\n        M\n       </tt>\n       indicates the section of the journal where the reference was published (e.g.,\n       <tt>\n        L\n       </tt>\n       for a letters section),\n       <tt>\n        PPPP\n       </tt>\n       gives the starting page number, and\n       <tt>\n        A\n       </tt>\n       is the first letter of the last name of the first author. Periods (\n       <tt>\n        .\n       </tt>\n       ) are used to fill unused fields and to pad fields out to their fixed length if too short; padding is done on the right for the publication code and on the left for the volume number and page number.\n       <sup class="reference" id="cite_ref-a_1-1">\n        <a href="#cite_note-a-1">\n         [1]\n        </a>\n       </sup>\n       <sup class="reference" id="cite_ref-b_2-1">\n        <a href="#cite_note-b-2">\n         [2]\n        </a>\n       </sup>\n       Page numbers greater than 9999 are continued in the\n       <tt>\n        M\n       </tt>\n       column. The 6-digit article ID numbers (in lieu of page numbers) used by the Physical Review publications since the late 1990s are treated as follows: The first two digits of the article ID, corresponding to the issue number, are converted to a lower-case letter (01 = a etc.) and inserted into column\n       <tt>\n        M\n       </tt>\n       . The remaining four digits are used in the page field.\n       <sup class="reference" id="cite_ref-b_2-2">\n        <a href="#cite_note-b-2">\n         [2]\n        </a>\n       </sup>\n      </p>\n      <h2>\n       <span class="mw-headline" id="Examples">\n        Examples\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=Bibcode&amp;action=edit&amp;section=3" title="Edit section: Examples">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <p>\n       Some examples of the code are:\n      </p>\n      <table class="wikitable">\n       <tr>\n        <th>\n         Bibcode\n        </th>\n        <th>\n         Reference\n        </th>\n       </tr>\n       <tr>\n        <td>\n         <tt>\n          <a class="external text" href="http://adsabs.harvard.edu/abs/1974AJ.....79..819H" rel="nofollow">\n           1974AJ.....79..819H\n          </a>\n         </tt>\n        </td>\n        <td>\n         <cite class="citation journal">\n          Heintz, W.
