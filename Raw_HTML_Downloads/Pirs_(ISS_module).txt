      <b>\n         Pirs\n        </b>\n       </i>\n       (\n       <a href="/wiki/Russian_language" title="Russian language">\n        Russian\n       </a>\n       :\n       <span lang="ru" xml:lang="ru">\n        \xd0\x9f\xd0\xb8\xd1\x80\xd1\x81\n       </span>\n       , meaning "\n       <a href="/wiki/Pier" title="Pier">\n        pier\n       </a>\n       ") -- also called "Stykovochny Otsek 1" or "SO-1" (\n       <a href="/wiki/Russian_language" title="Russian language">\n        Russian\n       </a>\n       :\n       <span lang="ru" xml:lang="ru">\n        \xd0\xa1\xd1\x82\xd1\x8b\xd0\xba\xd0\xbe\xd0\xb2\xd0\xbe\xd1\x87\xd0\xbd\xd1\x8b\xd0\xb9 \xd0\xbe\xd1\x82\xd1\x81\xd0\xb5\xd0\xba\n       </span>\n       , "\n       <a class="mw-redirect" href="/wiki/Docking_module" title="Docking module">\n        docking module\n       </a>\n       ", or DC-1 (docking compartment) -- is one of the two Russian docking compartments originally planned for the ISS.\n       <i>\n        Pirs\n       </i>\n       was launched in August 2001. It provides the ISS with one\n       <a class="mw-redirect" href="/wiki/Spacecraft_docking_and_berthing_mechanisms" title="Spacecraft docking and berthing mechanisms">\n        docking port\n       </a>\n       for Soyuz and Progress spacecraft, and allows egress and ingress for\n       <a class="mw-redirect" href="/wiki/Extra-vehicular_activity" title="Extra-vehicular activity">\n        spacewalks\n       </a>\n       by\n       <a href="/wiki/Astronaut" title="Astronaut">\n        cosmonauts\n       </a>\n       using Russian\n       <a href="/wiki/Orlan_space_suit" title="Orlan space suit">\n        Orlan space suits\n       </a>\n       .\n      </p>\n      <p>\n       A second docking compartment, "Stykovochniy Otsek 2" or SO-2, was initially planned with the same design. However, when the Russian segment of the ISS was redesigned in 2001, the new design no longer included the SO-2, and its construction was canceled.\n       <sup class="reference" id="cite_ref-1">\n        <a href="#cite_note-1">\n         [1]\n        </a>\n       </sup>\n       After another change of plans the SO-2 module finally evolved into the\n       <a href="/wiki/Poisk_(ISS_module)" title="Poisk (ISS module)">\n        <i>\n         Poisk\n        </i>\n       </a>\n       module, which was added to the ISS in 2009.\n      </p>\n      <p>\n       <i>\n        Pirs\n       </i>\n       was scheduled to be detached from the ISS and deorbited in 2017 by\n       <a href="/wiki/Progress_MS-06" title="Progress MS-06">\n        Progress MS-06\n       </a>\n       , to make room for the Russian\n       <a href="/wiki/Nauka_(ISS_module)" title="Nauka (ISS module)">\n        Multipurpose Laboratory Module\n        <i>\n         Nauka\n        </i>\n       </a>\n       , however due to the repetitive delays with the module, this maneuver was postponed for\n       <a href="/wiki/Progress_MS-09" title="Progress MS-09">\n        Progress MS-09\n       </a>\n       in 2018.\n     <a class="internal" href="/wiki/File:Iss017e011097.jpg" title="Enlarge">\n          </a>\n         </div>\n         Flight Engineer\n         <a href="/wiki/Oleg_Kononenko" title="Oleg The Docking Compartment is similar to the\n       <a href="/wiki/Mir_Docking_Module" title="Mir Docking Module">\n        Mir Docking Module\n       </a>\n       used on the\n       <a href="/wiki/Mir" title="Mir">\n        Mir\n       </a>\n       space station. The docking compartment\'s planned lifetime as part of the station was five years.\n      </p>\n      <h2>\n       <span class="mw-headline" id="Launch_in_2001">\n        Launch in 2001\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=Pirs_(ISS_module)&amp;action=edit&amp;section=2" title="Edit section: Launch in 2001">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <p>\n       <i>\n        Pirs\n       </i>\n       was launched on September 14, 2001, as ISS Assembly Mission 4R, on a Russian\n       <a href="/wiki/Soyuz-U" title="Soyuz-U">\n        Soyuz-U\n       </a>\n       rocket, using a modified\n       <a class="mw-redirect" href="/wiki/Progress_spacecraft" title="Progress spacecraft">\n        Progress spacecraft\n       </a>\n       ,\n       <a href="/wiki/Progress_M-SO1" title="Progress M-SO1">\n        Progress M-SO1\n       </a>\n       , as an upper stage.\n      </p>\n      <p>\n       The 3,580-kilogram Pirs Docking Compartment is attached to the nadir (bottom, Earth-facing) port of the\n       <a href="/wiki/Zvezda_(ISS_module)" title="Zvezda (ISS module)">\n        Zvezda\n       </a>\n       service module. It docked to the International Space Station on September 16, 2001, and was configured during three spacewalks by the\n       <a href="/wiki/Expedition_3" title="Expedition 3">\n        Expedition 3\n       </a>\n       crew. Two\n       <a href="/wiki/Strela_(crane)" title="Strela (crane)">\n        <i>\n         Strela\n        </i>\n       </a>\n       cargo cranes were later added by the\n       <a href="/wiki/STS-96" title="STS-96">\n        STS-96\n       </a>\n       and\n       <a href="/wiki/STS-101" title="STS-101">\n        STS-101\n       </a>\n       missions, carried up on\n       <a class="mw-redirect" href="/wiki/Integrated_Cargo_Carrier" title="Integrated Cargo Carrier">\n        Integrated Cargo Carriers\n       </a>\n       and installed during EVAs.\n      </p>\n      <h2>\n       <span class="mw-headline" id="Future">\n        Future\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a </span>\n       </span>\n      </h2>\n      <p>\n       <i>\n        Pirs\n       </i>\n       was originally scheduled to be detached from the nadir (bottom) port of the Zvezda module in 2014\n       <sup class="reference" id="cite_ref-2">\n        <a href="#cite_note-2">\n         [2]\n        </a>\n       </sup>\n       to make room for the Russian\n       <a class="mw-redirect" href="/wiki/Multipurpose_Laboratory_Module" title="Multipurpose Laboratory Module">\n        Multipurpose Laboratory Module\n       </a>\n       <i>\n        Nauka\n       </i>\n       .\n       <sup class="reference" id="cite_ref-NASA.gov_Consolidated_Launch_Manifest_3-0">\n        <a href="#cite_note-NASA.gov_Consolidated_Launch_Manifest-3">\n         [3]\n        </a>\n       </sup>\n       This has been delayed first to 2017 and then to 2018 due to a series of launch failures with the Proton rocket leading to delays in the launch of\n       <i>\n        Nauka\n       </i>\n       .\n       <sup class="reference" id="cite_ref-4">\n        <a href="#cite_note-4">\n         [4]\n        </a>\n       </sup>\n       When that happens,\n       <i>\n        Pirs\n       </i>\n       could become the first permanent ISS module to be decommissioned, and would be destroyed during atmospheric re-entry.\n      </p>\n      <h2>\n       <span class="mw-headline" id="Airlock_specifications">\n
