     </div>\n      </div>\n      <p>\n       In\n       <a href="/wiki/Physics" title="Physics">\n        physics\n       </a>\n       , a\n       <b>\n        rigid body\n       </b>\n       is a     deformation\n       </a>\n       is zero or so small it can be neglected. <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=Rigid_body&amp;action=edit&amp;section=1" title="Edit section: Kinematics">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <h3>\n       <span class="mw-headline" id="Linear_and_angular_position">\n        Linear and angular position\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=Rigid_body&amp;action=edit&amp;section=2" title="Edit section: Linear and angular position">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h3>\n      <p>\n       The position of a rigid body is the\n       <a href="/wiki/Coordinate_system#Examples" title="Coordinate system">\n        position\n       </a>\n       of all the particles of which it is composed. To simplify the description of this position, we exploit the property that the body is rigid, namely that all its particles maintain the same distance relative to each other. If the body is rigid, it is sufficient to describe the position of at least three non-\n       <a class="mw-redirect" href="/wiki/Collinear" title="Collinear">\n        collinear\n       </a>\n       particles. This makes it possible to reconstruct the position of all the other particles, provided that their\n       <a class="mw-redirect" href="/wiki/Time-invariant" title="Time-invariant">\n        time-invariant\n       </a>\n       position relative to the three selected particles is known. However, typically a different, mathematically more convenient, but equivalent approach is used. The position of the whole body is represented by:\n      </p>\n      <ol>\n       <li>\n        the\n        <b>\n         linear position\n        </b>\n        or\n        <b>\n         position\n        </b>\n        of the body, namely the position of one of the particles of the body, specifically chosen as a reference point (typically coinciding with the\n        <a href="/wiki/Center_of_mass" title="Center of mass">\n         center of mass\n        </a>\n        or\n        <a href="/wiki/Centroid" title="Centroid">\n         centroid\n        </a>\n        of the body), together with\n       </li>\n       <li>\n        the\n        <b>\n         <a href="/wiki/Orientation_(geometry)" title="Orientation (geometry)">\n          angular position\n         </a>\n        </b>\n        (also known as\n        <b>\n         orientation\n        </b>\n        , or\n        <b>\n         attitude\n        </b>\n        ) of the body.\n       </li>\n      </ol>\n      <p>\n       Thus, the position of a rigid body has two components:\n       <b>\n        linear\n       </b>\n       and\n       <b>\n        angular\n       </b>\n       , respectively.\n       <sup class="reference" id="cite_ref-2">\n        <a href="#cite_note-2">\n         [2]\n        </a>\n       </sup>\n       The same is true for other\n       <a href="/wiki/Kinematics" title="Kinematics">\n        kinematic\n       </a>\n       and\n       <a href="/wiki/Dynamics_(mechanics)" title="Dynamics (mechanics)">\n        kinetic\n       </a>\n       quantities describing the motion of a rigid body, such as linear and angular\n       <a href="/wiki/Velocity" title="Velocity">\n        velocity\n       </a>\n       ,\n       <a href="/wiki/Acceleration" title="Acceleration">\n        acceleration\n       </a>\n       ,\n       <a href="/wiki/Momentum" title="Momentum">\n        momentum\n       </a>\n       ,\n       <a href="/wiki/Impulse_(physics)" title="Impulse (physics)">\n        impulse\n       </a>\n       , and\n       <a href="/wiki/Kinetic_energy" title="Kinetic energy">\n        kinetic energy\n       </a>\n       .\n       <sup class="reference" id="cite_ref-3">\n        <a href="#cite_note-3">\n         [3]\n        </a>\n       </sup>\n      </p>\n      <p>\n       The linear\n       <a href="/wiki/Coordinate_system#Examples" title="Coordinate system">\n        position\n       </a>\n       can be represented by a\n       <a class="mw-redirect" href="/wiki/Position_vector" title="Position vector">\n        vector\n       </a>\n       with its tail at an arbitrary reference point in\n       <a href="/wiki/Space#Classical_mechanics" title="Space">\n        space\n       </a>\n       (the origin of a chosen\n       <a href="/wiki/Coordinate_system" title="Coordinate system">\n        coordinate system\n       </a>\n       ) and its tip at an arbitrary point of interest on the rigid body, typically coinciding with its\n       <a href="/wiki/Center_of_mass" title="Center of mass">\n        center of mass\n       </a>\n       or\n       <a href="/wiki/Centroid" title="Centroid">\n        centroid\n       </a>\n       . This reference point may define the origin of a\n       <a href="/wiki/Coordinate_system" title="Coordinate system">\n        coordinate system\n       </a>\n       fixed to the body.\n      </p>\n      <p>\n       There are several ways to numerically describe the\n       <a href="/wiki/Orientation_(geometry)" title="Orientation (geometry)">\n        orientation\n       </a>\n       of a rigid body, including a set of three\n       <a href="/wiki/Euler_angles" title="Euler angles">\n        Euler angles\n       </a>\n       , a\n       <a href="/wiki/Quaternion" title="Quaternion">\n        quaternion\n       </a>\n       , or a\n       <a class="mw-redirect" href="/wiki/Direction_cosines#Cartesian_coordinates" title="Direction cosines">\n        direction cosine matrix\n       </a>\n       (also referred to as a\n       <a href="/wiki/Rotation_matrix" title="Rotation matrix">\n        rotation matrix\n       </a>\n       ). All these methods actually define the orientation of a\n       <a href="/wiki/Basis_(linear_algebra)" title="Basis (linear algebra)">\n        basis set\n       </a>\n       (or\n       <a href="/wiki/Coordinate_system" title="Coordinate system">\n        coordinate system\n       </a>\n       ) which has a fixed orientation relative to the body (i.e. rotates together with the body), relative to another basis set (or coordinate system), from which the motion of the rigid body is observed. For instance, a basis set with fixed orientation relative to an airplane can be defined as a set of three orthogonal\n       <a href="/wiki/Unit_vector" title="Unit vector">\n        unit vectors\n b\n       </i>\n       <sub>\n        3\n       </sub>\n       , such that\n       <i>\n        b\n       </i>\n       <sub>\n        1\n       </sub>\n       is parallel to the chord line of the wing and directed forward,\n       <i>\n        b\n       </i>\n       <sub>\n        2\n       </sub>\n       is normal to the plane of symmetry and directed rightward, and\n       <i>\n        b\n       </i>\n       <sub>\n        3\n       </sub>\n       is given by the cross product\n       <span class="mwe-math-element">\n        <span class="mwe-math-mathml-inline mwe-math-mathml-a11y" height:2.509ex;"/>\n       </span>\n       .\n      </p>\n      <p>\n       In general, when a rigid body moves, both its position and orientation vary with time. In the kinematic sense, these changes are referred to as\n       <i>\n        <a class="mw-redirect" href="/wiki/Translation_(physics)" title="Translation (physics)">\n         translation\n        </a>\n       </i>\n       and\n       <i>\n        <a href="/wiki/Rotation" title="Rotation">\n         rotation\n        </a>\n       </i>\n       , respectively. Indeed, the position of a rigid body can be viewed as a hypothetic translation and rotation (roto-translation) of the body starting from a hypothetic reference position (not necessarily coinciding with a position actually taken by the body during its motion).\n      </p>\n      <h3>\n       <span class="mw-headline" id="Linear_and_angular_velocity">\n        Linear and angular velocity\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=Rigid_body&amp;action=edit&amp;section=3" title="Edit section: Linear and angular velocity">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h3>\n      <p>\n       <a href="/wiki/Velocity" title="Velocity">\n        Velocity\n       </a>\n       (also called\n       <b>\n        linear velocity\n       </b>\n       ) and\n       <a href="/wiki/Angular_velocity" title="Angular velocity">\n        angular velocity\n       </a>\n       are measured with respect to a\n       <a href="/wiki/Frame_of_reference" title="Frame of reference">\n        frame of reference\n       </a>\n       .\n      </p>\n      <p>\n       The linear\n       <b>\n        <a href="/wiki/Velocity" title="Velocity">\n         velocity\n        </a>\n       </b>\n       of a rigid body is a\n       <a class="mw-redirect" href="/wiki/Vector_(geometry)" title="Vector (geometry)">\n        vector\n       </a>\n       quantity, equal to the\n       <a href="/wiki/Time_derivative" title="Time derivative">\n        time rate of change\n       </a>\n       of its linear position. Thus, it is the velocity of a reference point fixed to the body. During purely translational motion (motion with no rotation), all points on a rigid body move with the same\n       <a href="/wiki/Velocity" title="Velocity">\n        velocity\n       </a>\n       . However, when\n       <a href="/wiki/Motion_(physics)" title="Motion (physics)">\n        motion\n       </a>\n       involves rotation, the instantaneous velocity of any two points on the body will generally not be the same. Two points of a rotating body will have the same instantaneous velocity only if they happen to lie on an axis parallel to the instantaneous\n       <a class="mw-redirect" href="/wiki/Axis_of_rotation" title="Axis of rotation">\n        axis of rotation\n       </a>\n       .\n      </p>\n      <p>\n       <b>\n        <a href="/wiki/Angular_velocity" title="Angular velocity">\n         Angular velocity\n        </a>\n       </b>\n       is a\n       <a class="mw-redirect" href="/wiki/Vector_(geometry)" title="Vector (geometry)">\n        vector\n       </a>\n       quantity that describes the\n       <a class="mw-redirect" href="/wiki/Angular_speed" title="Angular speed">\n        angular speed\n       </a>\n       at which the orientation of the rigid body is changing and the instantaneous\n       <a class="mw-redirect" href="/wiki/Axis_of_rotation" title="Axis of rotation">\n        axis\n       </a>\n       about which it is rotating (the existence of this instantaneous axis is guaranteed by the\n       <a href="/wiki/Euler%27s_rotation_theorem" title="Euler\'s rotation theorem">\n        Euler\'s rotation theorem\n       </a>\n       ). All points on a rigid body experience the same\n       <a href="/wiki/Angular_velocity" title="Angular velocity">\n        angular velocity\n       </a>\n       at all times. During purely rotational motion, all points on the body change position except for those lying on the instantaneous\n       <a class="mw-redirect" href="/wiki/Axis_of_rotation" title="Axis of rotation">\n        axis of rotation\n       </a>\n       . The relationship between orientation and angular velocity is not directly analogous to the relationship between position and velocity. Angular velocity is not the\n       <a href="/wiki/Time_derivative" title="Time derivative">\n        time rate of change\n       </a>\n       of orientation, because there is no such concept as an orientation vector that can be\n       <a href="/wiki/Derivative" title="Derivative">\n        differentiated\n       </a>\n       to obtain the angular velocity.\n      </p>\n      <h2>\n       <span the center of a\n        <a href="/wiki/Ball_and_socket_joint" title="Ball and socket joint">\n         ball and socket joint\n        </a>\n        , etc.\n       </li>\n      </ul>\n      <p>\n       When the center of mass is used as reference point:\n      </p>\n      <ul>\n       <li>\n        The (linear)\n        <a href="/wiki/Momentum" title="Momentum">\n         momentum\n        </a>\n        is independent of the rotational motion. At any time it is equal to the total mass of the rigid body times the translational velocity.\n       </li>\n       <li>\n        The\n        <a href="/wiki/Angular_momentum" title="Angular momentum">\n         angular momentum\n        </a>\n        with respect to the center of mass is the same as without translation: at any time it is equal to the\n        <a href="/wiki/Moment_of_inertia" title="Moment of inertia">\n         inertia tensor\n        </a>\n        times the angular velocity. When the angular velocity is expressed with respect to a coordinate system coinciding with the\n        <a href="/wiki/Moment_of_inertia#Inertia_tensor" title="Moment of inertia">\n         principal axes\n        </a>\n        of the body, each component of the angular momentum is a product of a moment of inertia (a principal value of the inertia tensor) times the corresponding component of the angular velocity; the\n        <a href="/wiki/Torque" title="Torque">\n         torque\n        </a>\n        is the inertia tensor times the\n        <a href="/wiki/Angular_acceleration" title="Angular acceleration">\n         angular acceleration\n        </a>\n        .\n       </li>\n       <li>\n        Possible motions in the absence of external forces are translation with constant velocity, steady rotation about a fixed principal axis, and also torque-free\n        <a href="/wiki/Precession" title="Precession">\n         precession\n        </a>\n        .\n       </li>\n       <li>\n        The net external force on the rigid body is always equal to the total mass times the translational acceleration (i.e.,\n        <a href="/wiki/Newton%27s_laws_of_motion"    </li>\n       <li>\n        The total\n        <a href="/wiki/Kinetic_energy" title="Kinetic energy">\n         kinetic energy\n        </a>\n        is simply the sum of translational and\n        <a href="/wiki/Rotational_energy" title="Rotational energy">\n         rotational energy\n        </a>\n        .\n       </li>\n      </ul>\n      <h2>\n       <span class="mw-headline" </span>\n       </span>\n      </h2>\n      <p>\n       Two rigid bodies are said to be\n       <a class="mw-redirect" href="/wiki/Equality_(objects)" title="Equality (objects)">\n        different\n       </a>\n       (not copies) if there is no\n       <a class="mw-redirect" href="/wiki/Proper_rotation" title="Proper rotation">\n        proper rotation\n       </a>\n       from one to the other. A rigid body is called\n       <a href="/wiki/Chirality_(mathematics)" title="Chirality (mathematics)">\n        chiral\n       </a>\n       if its\n       <a href="/wiki/Mirror_image" title="Mirror image">\n        mirror image\n       </a>\n       is different in that sense, i.e., if it has either no\n       <a href="/wiki/Symmetry" title="Symmetry">\n        symmetry\n       </a>\n       or its\n       <a href="/wiki/Symmetry_group" title="Symmetry group">\n        symmetry group\n       </a>\n       contains only proper rotations.
