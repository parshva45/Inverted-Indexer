ISSN\n       </b>\n       ) is an eight-digit\n       <a class="mw-redirect" href="/wiki/Serial_number" title="Serial number">\n        serial number\n       </a>\n       used to\n       <a href="/wiki/Unique_identifier" title="Unique identifier">\n        uniquely identify\n       </a>\n       a\n       <a href="/wiki/Serial_(publishing)" title="Serial (publishing)">\n        serial publication\n       </a>\n       .\n       <sup class="reference" id="cite_ref-Whatis_1-0">\n        <a href="#cite_note-Whatis-1">\n         [1]\n        </a>\n       </sup>\n       The ISSN is especially helpful in distinguishing between serials with the same title. ISSN are used in ordering, cataloging, interlibrary loans, and other practices in connection with serial literature.\n       <sup class="reference" id="cite_ref-2">\n        <a href="#cite_note-2">\n         [2]\n        </a>\n       </sup>\n      </p>\n      <p>\n       The ISSN system was first drafted as an\n       <a href="/wiki/International_Organization_for_Standardization" title="International Organization for Standardization">\n        International Organization for Standardization\n       </a>\n       (ISO)\n       <a href="/wiki/International_standard" title="International standard">\n        international standard\n       </a>\n       in 1971 and published as ISO 3297 in 1975.\n       <sup class="reference" id="cite_ref-3">\n        <a href="#cite_note-3">\n         [3]\n        </a>\n       </sup>\n       ISO subcommittee\n       <a href="/wiki/TC_46/SC_9" title="TC 46/SC 9">\n        TC 46/SC 9\n       </a>\n       is responsible for maintaining the standard.\n      </p>\n      <p>\n       When a serial with the same\n       <a href="/wiki/Content_(media)" title="Content (media)">\n        content\n       </a>\n       is published in more than one\n       <a href="/wiki/Media_(communication)" title="Media (communication)">\n        media type\n       </a>\n       , a different ISSN is assigned to each media type. For example, many serials are published both in\n       <a href="/wiki/Printing" title="Printing">\n        print\n       </a>\n       and\n       <a href="/wiki/Electronic_media" title="Electronic media">\n        electronic media\n       </a>\n       . The ISSN system refers to these types as\n       <b>\n        print ISSN\n       </b>\n       (\n       <b>\n        p-ISSN\n       </b>\n       ) and\n       <b>\n        electronic ISSN\n       </b>\n       (\n       <b>\n        e-ISSN\n       </b>\n       ), respectively.\n       <sup class="reference" id="cite_ref-4">\n        <a href="#cite_note-4">\n         [4]\n        </a>\n       </sup>\n       Conversely, as defined in ISO 3297:2007, every serial in the ISSN system is also assigned a\n       <b>\n        linking ISSN\n       </b>\n       (\n       <b>\n        ISSN-L\n       </b>\n       ), typically the same as the ISSN assigned to the serial in its first published medium, which links together all ISSNs assigned to the serial in every medium.\n       <sup class="reference" id="cite_ref-5">\n        <a href="#cite_note-5">\n         [5]\n        </a>\n       </sup>\n      </p>\n      <p>\n      </p>\n      <div class="toc" id="toc">\n       <div <p>\n      </p>\n      <h2>\n       <span class="mw-headline" id="Code_format">\n        Code format\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Standard_Serial_Number&amp;action=edit&amp;section=1" title="Edit section: Code format">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <p>\n       The format of the ISSN is an eight digit code, divided by a hyphen into two four-digit numbers.\n       <sup class="reference" id="cite_ref-Whatis_1-1">\n        <a href="#cite_note-Whatis-1">\n         [1]\n        </a>\n       </sup>\n       As an\n       <a class="mw-redirect" href="/wiki/Integer_number" title="Integer number">\n        integer number\n       </a>\n       , it can be represented by the first seven digits.\n       <sup class="reference" id="cite_ref-6">\n        <a href="#cite_note-6">\n         [6]\n        </a>\n       </sup>\n       The last code digit, which may be 0-9 or an X, is a\n       <a href="/wiki/Check_digit" title="Check digit">\n        check digit\n       </a>\n       . Formally, the general form of the ISSN code (also named "ISSN structure" or "ISSN syntax") can be expressed as follows:\n       <sup class="reference" id="cite_ref-7">\n        <a href="#cite_note-7">\n         [7]\n        </a>\n       </sup>\n      </p>\n      <dl>\n       <dd>\n        <code class="mw-highlight" dir="ltr" id="" style="">\n         NNNN-NNNC\n the check digit is X, then add 10 to the sum). The modulus 11 of the sum must be 0.\n      </p>\n      <p>\n       There is an online ISSN checker that can validate an ISSN, based on the above algorithm.\n       <sup class="reference" id="cite_ref-9">\n        <a href="#cite_note-9">\n         [9]\n        </a>\n       </sup>\n       <sup class="reference" id="cite_ref-10">\n        <a href="#cite_note-10">\n         [10]\n        </a>\n       </sup>\n      </p>\n      <h2>\n       <span class="mw-headline" id="Code_assignment">\n        Code assignment\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Standard_Serial_Number&amp;action=edit&amp;section=2" title="Edit section: Code assignment">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <p>\n       ISSN codes are assigned by a network of ISSN National Centres, usually located at\n       <a href="/wiki/National_library" title="National library">\n        national libraries\n       </a>\n       and coordinated by the\n       <b>\n        <a class="mw-redirect" href="/wiki/ISSN_International_Centre" title="ISSN International Centre">\n         ISSN International Centre\n        </a>\n       </b>\n       based in\n       <a href="/wiki/Paris" title="Paris">\n        Paris\n       </a>\n       . The International Centre is an\n       <a href="/wiki/International_organization" title="International organization">\n        intergovernmental organization\n       </a>\n       created in 1974 through an agreement between\n       <a href="/wiki/UNESCO" title="UNESCO">\n        UNESCO\n       </a>\n       and the French government. The International Centre maintains a database of all ISSNs assigned worldwide, the\n       <i>\n        ISDS Register\n       </i>\n       (International Serials Data System) otherwise known as the\n       <i>\n        ISSN Register\n       </i>\n       . At the end of 2016\n       <sup class="plainlinks noprint asof-tag update" style="display:none;">\n        <a class="external text" href="//en.wikipedia.org/w/index.php?title=International_Standard_Serial_Number&amp;action=edit">\n         [update]\n        </a>\n       </sup>\n       , the ISSN Register contained records for 1,943,572 items.\n       <sup class="reference" id="cite_ref-11">\n        <a href="#cite_note-11">\n         [11]\n        </a>\n       </sup>\n      </p>\n      <h2>\n       <span class="mw-headline" id="Comparison_with_other_identifiers">\n        Comparison with other identifiers\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Standard_Serial_Number&amp;action=edit&amp;section=3" title="Edit section: Comparison with other identifiers">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <p>\n       ISSN and\n       <a class="mw-redirect" href="/wiki/ISBN" title="ISBN">\n        ISBN\n       </a>\n       codes are similar in concept, where ISBNs are assigned to individual\n       <a href="/wiki/Book" title="Book">\n        books\n       </a>\n       . An ISBN might be assigned for particular issues of a serial, in addition to the ISSN code for the serial as a whole. An ISSN, unlike the ISBN code, is an anonymous identifier associated with a serial title, containing no\n       <a href="/wiki/Information" title="Information">\n        information\n       </a>\n       as to the\n       <a class="mw-redirect" href="/wiki/Publisher" title="Publisher">\n        publisher\n       </a>\n       or its\n       <a href="/wiki/Location_(geography)" title="Location (geography)">\n        location\n       </a>\n       . For this reason a new ISSN is assigned to a serial each time it undergoes a major title change.\n      </p>\n      <p>\n       Since the ISSN applies to an entire serial a new identifier, the\n       <a href="/wiki/Serial_Item_and_Contribution_Identifier" title="Serial Item and Contribution Identifier">\n        Serial Item and Contribution Identifier\n       </a>\n       (SICI), was built on top of it to allow references to specific volumes, articles, or other identifiable components (like the\n       <a href="/wiki/Table_of_contents" title="Table of contents">\n        table of contents\n       </a>\n       ).\n      </p>\n      <h3>\n       <span class="mw-headline" id="Media_vs_content">\n        Media\n        <i>\n         vs\n        </i>\n        content\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Standard_Serial_Number&amp;action=edit&amp;section=4" title="Edit section: Media vs content">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h3>\n      <p>\n       Separate ISSNs are needed for serials in different media (except reproduction\n       <a href="/wiki/Microform" title="Microform">\n        microforms\n       </a>\n       ). Thus, the\n       <a class="mw-redirect" href="/wiki/Print_media" title="Print media">\n        print\n       </a>\n       and\n       <a href="/wiki/Electronic_media" title="Electronic media">\n        electronic media\n       </a>\n       versions of a serial need separate ISSNs.\n       <sup class="reference" id="cite_ref-12">\n        <a href="#cite_note-12">\n         [12]\n        </a>\n       </sup>\n       Also, a\n       <a href="/wiki/CD-ROM" title="CD-ROM">\n        CD-ROM\n       </a>\n       version and a\n       <a href="/wiki/World_Wide_Web" title="World Wide Web">\n        web\n       </a>\n       version of a serial require different ISSNs since two different media are involved. However, the same ISSN can be used for different file formats (e.g.\n       <a class="mw-redirect" href="/wiki/PDF" title="PDF">\n        PDF\n       </a>\n       and\n       <a href="/wiki/HTML" title="HTML">\n        HTML\n       </a>\n       ) of the same online serial.\n      </p>\n      <p>\n       This "media-oriented identification" of serials made sense in the 1970s. In the 1990s and onward, with personal computers, better screens, and the Web, it makes sense to consider only\n       <i>\n        content\n       </i>\n       , independent of media. This "content-oriented identification" of serials was a\n       <a class="mw-redirect" href="/wiki/Excess_demand" title="Excess demand">\n        repressed demand\n       </a>\n       during a decade, but no ISSN update or initiative occurred. A natural extension for ISSN, the unique-identification of the articles in the serials, was the main demand application. An alternative serials\' contents model arrived with the\n       <a href="/wiki/Indecs_Content_Model" title="Indecs Content Model">\n        indecs Content Model\n       </a>\n       and its application, the\n       <a href="/wiki/Digital_object_identifier" title="Digital object identifier">\n        digital object identifier\n       </a>\n       (DOI), as ISSN-independent initiative, consolidated in the 2000s.\n      </p>\n      <p>\n       Only later, in 2007, ISSN-L was defined in the new ISSN standard (ISO 3297:2007) as an "ISSN designated by the ISSN Network to enable collocation or versions of a continuing resource linking among the different media".\n       <sup class="reference" id="cite_ref-issnL_13-0">\n        <a href="#cite_note-issnL-13">\n         [13]\n        </a>\n       </sup>\n      </p>\n      <h2>\n       <span class="mw-headline" id="Availability">\n        Availability\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Standard_Serial_Number&amp;action=edit&amp;section=5" title="Edit section: Availability">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <p>\n       The ISSN Register is not freely available for interrogation on the web, but is available by subscription. There are several routes to the identification and verification of ISSN codes for the public:\n      </p>\n      <ul>\n       <li>\n        The print version of a serial typically will include the ISSN code as part of the publication information.\n       </li>\n       <li>\n        Most serial websites contain ISSN code information.\n       </li>\n       <li>\n        Derivative lists of publications will often contain ISSN codes; these can be found through on-line searches with the ISSN code itself or serial title.\n       </li>\n       <li>\n        <a href="/wiki/WorldCat" title="WorldCat">\n         WorldCat\n        </a>\n        permits searching its catalog by ISSN, by entering "issn:"+ISSN code in the query field. One can also go directly to an ISSN\'s record by appending it to "\n        <code class="mw-highlight" dir="ltr" id="" style="">\n         https://www.worldcat.org/ISSN/\n        </code>\n        ", e.g.\n        <a class="external free" href="https://www.worldcat.org/ISSN/1021-9749" rel="nofollow">\n         https://www.worldcat.org/ISSN/1021-9749\n        </a>\n        . This does not query the ISSN Register itself, but rather shows whether any Worldcat library holds an item with the given ISSN.\n       </li>\n      </ul>\n      <h2>\n       <span class="mw-headline" id="Use_in_URNs">\n        Use in URNs\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Standard_Serial_Number&amp;action=edit&amp;section=6" title="Edit section: Use in URNs">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <p>\n       An ISSN can be encoded as a\n       <a class="mw-redirect" href="/wiki/Uniform_resource_name" title="Uniform resource name">\n        uniform resource name\n       </a>\n       (URN) by prefixing it with "\n       <code class="mw-highlight" dir="ltr" id="" style="">\n        urn:ISSN:\n       </code>\n       ".\n       <sup class="reference" id="cite_ref-14">\n        <a href="#cite_note-14">\n         [14]\n        </a>\n       </sup>\n       For example,\n       <a href="/wiki/Rail_(magazine)" title="Rail (magazine)">\n        <i>\n         Rail\n        </i>\n       </a>\n       could be referred to as "\n       <code class="mw-highlight" dir="ltr" id="" style="">\n        urn:ISSN:0953-4563\n       </code>\n       ". URN namespaces are case-sensitive, and the ISSN namespace is all caps.\n       <sup class="reference" id="cite_ref-15">\n        <a href="#cite_note-15">\n         [15]\n        </a>\n       </sup>\n       If the checksum digit is "X" then it is always encoded in uppercase in a URN.\n      </p>\n      <h3>\n       <span class="mw-headline" id="Problems">\n        Problems\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Standard_Serial_Number&amp;action=edit&amp;section=7" title="Edit section: Problems">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h3>\n      <p>\n       The util URNs are\n       <a href="#Media_vs_Content">\n        content-oriented\n       </a>\n       , but ISSN is media-oriented:\n      </p>\n      <ul>\n       <li>\n        <b>\n         ISSN is not unique\n        </b>\n        when the concept is "a journal is a set of contents, generally copyrighted content": the same journal (same contents and same copyrights) have two or more ISSN codes. A URN needs to point to "unique content" (a "unique journal" as a "set of contents" reference).\n        <br/>\n        Examples:\n        <a href="/wiki/Nature_(journal)" title="Nature (journal)">\n         Nature\n        </a>\n        has an ISSN for print, 0028-0836, and another for the same content on the Web, 1476-4687; only the oldest (0028-0836) is used as a\n        <a href="/wiki/Unique_identifier" title="Unique identifier">\n         unique identifier\n        </a>\n        . As the ISSN is not unique, the\n        <a class="mw-redirect" href="/wiki/U.S._National_Library_of_Medicine" title="U.S. National Library of Medicine">\n         U.S. National Library of Medicine\n        </a>\n        needed to create, prior to 2007, the NLM Unique ID (JID).\n        <sup class="reference" id="cite_ref-16">\n         <a href="#cite_note-16">\n          [16]\n         </a>\n        </sup>\n       </li>\n       <li>\n        <b>\n         ISSN\n        </b>\n        does not offer\n        <a class="mw-redirect" href="/wiki/Uniform_resource_name#Resolvers" title="Uniform resource name">\n         resolution mechanisms\n        </a>\n        like a\n        <a href="/wiki/Digital_object_identifier" title="Digital object identifier">\n         digital object identifier\n        </a>\n        (DOI) or an URN does, so the DOI is used as a URN for articles, with (for historical reasons) no need for an ISSN\'s existence.\n        <br/>\n        Example: the DOI name "10.1038/nature13777" can be represented as an HTTP string by\n        <code>\n         <i>\n          https://dx.doi.org/\n         </i>\n         10.1038/nature13777\n        </code>\n        , and is\n        <a href="/wiki/URL_redirection" title="URL redirection">\n         redirected\n        </a>\n        (resolved) to the current article\'s page; but there is no ISSN online service, like\n        <code>\n         http://dx.issn.org/\n        </code>\n        , to resolve the ISSN of the journal (in this sample\n        <a href="/wiki/Nature_(journal)" title="Nature (journal)">\n         1476-4687\n        </a>\n        ), that is, a kind of\n        <code>\n         <i>\n          https://dx.issn.org/\n         </i>\n         1476-4687\n        </code>\n        redirecting to the journal\'s home.\n       </li>\n      </ul>\n      <p>\n       A unique URN for serials simplifies the search, recovery and delivery of data for various services including, in particular,\n       <a href="/wiki/Information_retrieval" title="Information retrieval">\n        search systems\n       </a>\n       and\n       <a href="/wiki/Knowledge_base" title="Knowledge base">\n        knowledge databases\n       </a>\n       .\n       <sup class="reference" id="cite_ref-issnL_13-1">\n        <a href="#cite_note-issnL-13">\n         [13]\n        </a>\n       </sup>\n       <b>\n        ISSN-L\n       </b>\n       (see\n       <a href="#Linking_ISSN">\n        Linking ISSN\n       </a>\n       below) was created to fill this gap.\n      </p>\n      <h2>\n       <span class="mw-headline" id="ISSN_variants_and_labels">\n        ISSN variants and labels\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Standard_Serial_Number&amp;action=edit&amp;section=8" title="Edit section: ISSN variants and labels">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <p>\n       There are two most popular media types that adopted\n       <i>\n        special labels\n       </i>\n       (indicating below in italics), and one in fact ISSN-variant, with also an optional label. All are used in standard\n       <a href="/wiki/Metadata" title="Metadata">\n        metadata\n       </a>\n       context like\n       <a class="mw-redirect" href="/wiki/JATS" title="JATS">\n        JATS\n       </a>\n       , and the labels also, frequently, as abbreviations.\n      </p>\n      <h3>\n       <span class="mw-headline" id="Print_ISSN">\n        Print ISSN\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Standard_Serial_Number&amp;action=edit&amp;section=9" title="Edit section: Print ISSN">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h3>\n      <p>\n       <i>\n        p-ISSN\n       </i>\n       is a standard label for "Print ISSN", the ISSN for the\n       <a class="mw-redirect" href="/wiki/Print_media" title="Print media">\n        print media\n       </a>\n       (paper) version of a serial. Usually it is the "default media", so the "default ISSN".\n      </p>\n      <h3>\n       <span class="mw-headline" id="Electronic_ISSN">\n        Electronic ISSN\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Standard_Serial_Number&amp;action=edit&amp;section=10" title="Edit section: Electronic ISSN">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h3>\n      <p>\n       <i>\n        e-ISSN\n       </i>\n       (or eISSN) is a standard label for "Electronic ISSN", the ISSN for the\n       <a href="/wiki/Electronic_media" title="Electronic media">\n        electronic media\n       </a>\n       (online) version of a serial.\n      </p>\n      <h3>\n       <span class="mw-headline" id="Linking_ISSN">\n        Linking ISSN\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Standard_Serial_Number&amp;action=edit&amp;section=11" title="Edit section: Linking ISSN">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h3>\n      <p>\n       <i>\n        ISSN-L\n       </i>\n       is a\n       <a href="/wiki/Unique_identifier" title="Unique identifier">\n        unique identifier\n       </a>\n       for all versions of the serial containing the same\n       <a href="/wiki/Content_(media)" title="Content (media)">\n        content\n       </a>\n       across different media. As defined by\n       <i>\n        ISO 3297:2007\n       </i>\n       , the "linking ISSN (ISSN-L)" provides a mechanism for collocation or linking among the different media versions of the same continuing resource.\n      </p>\n      <p>\n       The\n       <i>\n        ISSN-L\n       </i>\n       is one ISSN number among the existing ISSNs, so, does not change the use or assignment of "ordinary" ISSNs;\n       <sup class="reference" id="cite_ref-17">\n        <a href="#cite_note-17">\n         [17]\n        </a>\n       </sup>\n       it is based on the ISSN of the first published medium version of the publication. If the print and online versions of the publication are published at the same time, the ISSN of the print version is chosen as the basis of the\n       <i>\n        ISSN-L\n       </i>\n       .\n      </p>\n      <p>\n       With\n       <i>\n        ISSN-L\n       </i>\n       is possible to designate one single ISSN for all those media versions of the title. The use of\n       <i>\n        ISSN-L\n       </i>\n       facilitates search, retrieval and delivery across all media versions for services like\n       <a href="/wiki/OpenURL" title="OpenURL">\n        OpenURL\n       </a>\n       ,\n       <a class="mw-redirect" href="/wiki/Library_catalogue" title="Library catalogue">\n        library catalogues\n       </a>\n       ,\n       <a class="mw-redirect" href="/wiki/Search_engine" title="Search engine">\n        search engines\n       </a>\n       or\n       <a href="/wiki/Knowledge_base" title="Knowledge base">\n        knowledge bases\n       </a>\n       .\n       <sup class="reference" id="cite_ref-18">\n        <a href="#cite_note-18">\n         [18]\n        </a>\n       </sup>\n      </p>\n      <h2>\n       <span class="mw-headline" id="ROAD">\n        ROAD\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Standard_Serial_Number&amp;action=edit&amp;section=12" title="Edit section: ROAD">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <ul>\n       <li>\n        <a class="new" href="/w/index.php?title=ROAD:_Directory_of_Open_Access_Scholarly_Resources&amp;action=edit&amp;redlink=1" title="ROAD: Directory of Open Access Scholarly Resources (page does not exist)">\n         ROAD: Directory of Open Access Scholarly Resources\n        </a>\n        <span class="noprint" style="font-size:85%;
