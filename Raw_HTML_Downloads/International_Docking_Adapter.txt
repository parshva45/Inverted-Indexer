</div>\n      <p>\n       The\n       <b>\n        International Docking Adapter\n       </b>\n       (\n       <b>\n        IDA\n       </b>\n       ) is a\n       <a href="/wiki/Docking_and_berthing_of_spacecraft#Adapters" title="Docking and berthing of spacecraft">\n        spacecraft docking system adapter\n       </a>\n       being developed to convert\n       <a href="/wiki/Androgynous_Peripheral_Attach_System#APAS-95" title="Androgynous Peripheral Attach System">\n        APAS-95\n       </a>\n       to the\n       <a href="/wiki/NASA_Docking_System" title="NASA Docking System">\n        NASA Docking System\n       </a>\n       (NDS)/\n       <a href="/wiki/International_Docking_System_Standard" title="International Docking System Standard">\n        International Docking System Standard\n       </a>\n       (IDSS). An IDA will be placed on each of the\n       <a href="/wiki/International_Space_Station" title="International Space Station">\n        International Space Station\'s\n       </a>\n       (ISS) two open\n       <a class="mw-redirect" href="/wiki/Pressurized_Mating_Adapters" title="Pressurized Mating Adapters">\n        Pressurized Mating Adapters\n       </a>\n       (PMA), both of which will be connected to Node-2 (\n       <a href="/wiki/Harmony_(ISS_module)" title="Harmony (ISS module)">\n        Harmony module\n       </a>\n       ).\n      </p>\n      <p>\n      </p>\n      <div        </span>\n         </a>\n        </li>\n       </ul>\n      </div>\n      <p>\n      </p>\n      <h2>\n       <span class="mw-headline" id="History">\n        History\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Docking_Adapter&amp;action=edit&amp;section=1" title="Edit section: History">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <p>\n       Prior to IDA there were several different iterations of docking adapters that were meant to fill a similar role but were ultimately canceled.\n      </p>\n      <h3>\n       <span class="mw-headline" id="APAS_to_LIDS_Adaptor_System">\n        APAS to LIDS Adaptor System\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Docking_Adapter&amp;action=edit&amp;section=2" title="Edit section: APAS to LIDS Adaptor System">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h3>\n      <p>\n       The APAS to LIDS Adaptor System (ATLAS) was announced in 2008.\n       <sup class="reference" id="cite_ref-flightglobal_1-0">\n        <a href="#cite_note-flightglobal-1">\n         [1]\n        </a>\n       </sup>\n       It would have been placed on the open PMAs and converted APAS-95 to the\n       <a class="mw-redirect" href="/wiki/Low_Impact_Docking_System" title="Low Impact Docking System">\n        Low Impact Docking System\n       </a>\n       (LIDS).\n       <sup class="reference" id="cite_ref-flightglobal_1-1">\n        <a href="#cite_note-flightglobal-1">\n         [1]\n        </a>\n       </sup>\n       ATLAS was planned to be launched on\n       <a href="/wiki/Orion_(spacecraft)" title="Orion (spacecraft)">\n        Orion\'s\n       </a>\n       first two missions to the International Space Station.\n       <sup class="reference" id="cite_ref-flightglobal_1-2">\n        <a href="#cite_note-flightglobal-1">\n         [1]\n        </a>\n       </sup>\n       Orion\'s missions to the ISS were later canceled altogether and its role as a crew transporter was replaced by the\n       <a class="mw-redirect" href="/wiki/Commercial_Crew_Program" title="Commercial Crew Program">\n        Commercial Crew Program\n       </a>\n       .\n       <sup class="reference" id="cite_ref-2">\n        <a href="#cite_note-2">\n         [2]\n        </a>\n       </sup>\n      </p>\n      <h3>\n       <span class="mw-headline" id="Common_Docking_Adapter">\n        Common Docking Adapter\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Docking_Adapter&amp;action=edit&amp;section=3" title="Edit section: Common Docking Adapter">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h3>\n      <div class="thumb tright">\n       <div class="thumbinner" style="width:222px;">\n        <a class="image" href="/wiki/File:Common_Docking_Adapter.png">\n         <img alt="" class="thumbimage" data-file-height="200" data-file-width="315" height="140" src="//upload.wikimedia.org/wikipedia/commons/thumb/7/7f/Common_Docking_Adapter.png/220px-Common_Docking_Adapter.png" srcset="//upload.wikimedia.org/wikipedia/commons/7/7f/Common_Docking_Adapter.png 1.5x" width="220"/>\n        </a>\n        <div class="thumbcaption">\n         <div class="magnify">\n          <a class="internal" href="/wiki/File:Common_Docking_Adapter.png" title="Enlarge">\n          </a>\n         </div>\n         Rendering of the now-cancelled Common Docking Adapter\n        </div>\n       </div>\n      </div>\n      <p>\n       The Common Docking Adapter (CDA) was announced in 2009.\n       <sup class="reference" id="cite_ref-3">\n        <a href="#cite_note-3">\n         [3]\n        </a>\n       </sup>\n       It was planned to be used to convert the\n       <a href="/wiki/Common_Berthing_Mechanism" title="Common Berthing Mechanism">\n        Common Berthing Mechanism\n       </a>\n       to the NASA Docking System.\n       <sup class="reference" id="cite_ref-CDA_4-0">\n        <a href="#cite_note-CDA-4">\n         [4]\n        </a>\n       </sup>\n       The CDAs would have been attached directly to Node-2 forward and Node-2 zenith.\n       <sup class="reference" id="cite_ref-CDA_4-1">\n        <a href="#cite_note-CDA-4">\n         [4]\n        </a>\n       </sup>\n       They would have been delivered on the Japanese\n       <a href="/wiki/H-II_Transfer_Vehicle" title="H-II Transfer Vehicle">\n        HTV cargo spacecraft\n       </a>\n       .\n       <sup class="reference" id="cite_ref-CDA_4-2">\n        <a href="#cite_note-CDA-4">\n         [4]\n        </a>\n       </sup>\n      </p>\n      <h2>\n       <span class="mw-headline" id="Design">\n        Design\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Docking_Adapter&amp;action=edit&amp;section=4" title="Edit section: Design">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <p>\n       IDA converts APAS-95 to the NASA Docking System (which complies with the International Docking System Standard) and allows the transfer of crew, cargo, power and data.\n       <sup class="reference" id="cite_ref-Siceloff_5-0">\n        <a href="#cite_note-Siceloff-5">\n         [5]\n        </a>\n       </sup>\n       IDA has a mass of 526\xc2\xa0kg (1,160\xc2\xa0lb), a height of 110\xc2\xa0cm (42\xc2\xa0in) and a width of 160\xc2\xa0cm (63\xc2\xa0in).\n       <sup class="reference" id="cite_ref-Siceloff_5-1">\n        <a href="#cite_note-Siceloff-5">\n         [5]\n        </a>\n       </sup>\n       <sup class="reference" id="cite_ref-6">\n        <a href="#cite_note-6">\n         [6]\n        </a>\n       </sup>\n       When including the docking targets, laser retro-reflectors and related systems that are arrayed around the outer perimeters, the outer diameter is about 240\xc2\xa0cm (94\xc2\xa0in).\n       <sup class="reference" id="cite_ref-Siceloff_5-2">\n        <a href="#cite_note-Siceloff-5">\n         [5]\n        </a>\n       </sup>\n      </p>\n      <p>\n       <a href="/wiki/Boeing" title="Boeing">\n        Boeing\n       </a>\n       is the primary contractor for the IDAs and the adapters were assembled at their\n       <a href="/wiki/Houston" title="Houston">\n        Houston\n       </a>\n       Product Support Center.\n       <sup class="reference" id="cite_ref-7">\n        <a href="#cite_note-7">\n         [7]\n        </a>\n       </sup>\n       <sup class="reference" id="cite_ref-NASAABC_8-0">\n        <a href="#cite_note-NASAABC-8">\n         [8]\n        </a>\n       </sup>\n       Parts from subcontractors came from 25 American states and the primary structure is from Russian company\n       <a class="mw-redirect" href="/wiki/RSC_Energia" title="RSC Energia">\n        RSC Energia\n       </a>\n       .\n       <sup class="reference" id="cite_ref-Siceloff_5-3">\n        <a href="#cite_note-Siceloff-5">\n         [5]\n        </a>\n       </sup>\n       <sup class="reference" id="cite_ref-NASAABC_8-1">\n        <a href="#cite_note-NASAABC-8">\n         [8]\n        </a>\n       </sup>\n      </p>\n      <h2>\n       <span class="mw-headline" id="Deployment_of_IDA_modules">\n        Deployment of IDA modules\n       </span>\n       <span class="mw-editsection">\n        <span class="mw-editsection-bracket">\n         [\n        </span>\n        <a href="/w/index.php?title=International_Docking_Adapter&amp;action=edit&amp;section=5" title="Edit section: Deployment of IDA modules">\n         edit\n        </a>\n        <span class="mw-editsection-bracket">\n         ]\n        </span>\n       </span>\n      </h2>\n      <p>\n       When each IDA arrives,\n       <a href="/wiki/Dextre" title="Dextre">\n        Dextre\n       </a>\n       removes it from Dragon\'s trunk and moves it to about 30\xc2\xa0cm (1\xc2\xa0ft) from the front of the PMA. It then moves the IDA carefully into position until it is seated on the PMA and holds it there.\n       <sup class="reference" id="cite_ref-NASAABC_8-2">\n        <a href="#cite_note-NASAABC-8">\n         [8]\n        </a>\n       </sup>\n       Astronauts during an\n       <a href="/wiki/Extravehicular_activity" title="Extravehicular activity">\n        Extravehicular activity\n       </a>\n       then complete the electrical connections and permanently connect it to the PMA.\n       <sup class="reference" id="cite_ref-NASAABC_8-3">\n        <a href="#cite_note-NASAABC-8">\n         [8]\n        </a>\n       </sup>\n      </p>\n      <p>\n       IDA-1 was planned to be attached to Node-2\'s forward PMA.\n       <sup class="reference" id="cite_ref-Hartman_9-0">\n        <a href="#cite_note-Hartman-9">\n         [9]\n        </a>\n       </sup>\n       IDA-2 was originally planned to be attached to Node-2\'s zenith PMA.\n       <sup class="reference" id="cite_ref-Hartman_9-1">\n        <a href="#cite_note-Hartman-9">\n         [9]\n        </a>\n       </sup>\n       <sup class="reference" id="cite_ref-launchdate_10-0">\n        <a
