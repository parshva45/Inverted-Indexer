The module is a new generation module which can replace the Zarya Functional Cargo Block (FGB) originally used in the\n       <a class="mw-redirect" href="/wiki/TKS_spacecraft" title="TKS spacecraft">\n        TKS spacecraft\n       </a>\n       <sup class="reference" id="cite_ref-2">\n        <a href="#cite_note-2">\n         [2]\n        </a>\n       </sup>\n       which was the
